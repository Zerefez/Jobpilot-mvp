# Resume Coach - Quick Start (3 Steps)

## 1. Get OpenAI API Key
Visit https://platform.openai.com/api-keys and create a new API key

## 2. Create `.env.local`
In the `jobpilot` folder, create a file named `.env.local`:
```
OPENAI_API_KEY=sk-your_key_here
```

## 3. Install & Run
```bash
cd jobpilot
npm install
npm run dev
```

## 4. Open in Browser
Go to: http://localhost:3000/resume-coach

## Usage
1. **Left Side**: Paste/upload your job description, resume, and cover letter
2. **Click**: "Start Analysis" button
3. **Right Side**: Chat with AI to get recommendations and improved drafts
4. **Iterate**: Keep asking questions until you're satisfied!

---

**That's it!** You now have a working AI resume coach running locally.

For more details, see `RESUME_COACH_SETUP.md`
